2.安装hexo

2.注册GitHub账号

注册好后创建项目，项目名称为username+.github.io

然后在保留博客本地文件的界面打开git

cpnm install –save hexo-deployer-git

hexo clean 清理一下

hexo g 重新生成内容

hexo d 部署
